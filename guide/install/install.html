# 项目下载和运行

  • 安装依赖
yarn install
  • 开发模式运行
yarn run serve
  • 编译项目
yarn run build
  • Lints and fixes files
yarn run lint

# 其他说明

  • 项目使用的 vue-cli3, 请更新您的 cli

  • 关闭 Eslint (不推荐) 移除 package.json 中 eslintConfig 整个节点代码

  • 修改 Ant Design 配色，在文件 vue.config.js 中，其他 less 变量覆盖参考 ant design 官方说明

  css: {
    loaderOptions: {
      less: {
        modifyVars: {
          /* less 变量覆盖，用于自定义 ant design 主题 */

          'primary-color': '#F5222D',
          'link-color': '#F5222D',
          'border-radius-base': '4px',
        },
        javascriptEnabled: true,
      }
    }
  }

# 目录结构

├── dist                       // 打包后的目录
├── src                        // 源代码
│   ├── api                    // 所有请求
│   ├── assets                 // 主题 字体等静态资源
│   ├── components             // 组件集合
│        ├── _util             // 组件通用方法
│        ├── AvatarList        // 组件通用方法
│   ├── plugins                // 插件
│   ├── config                 // 配置信息
│        └── router.config.js  // 路由配置信息
│   ├── mixins                 // 混入对象
│   ├── router                 // 路由对象
│   ├── store                  // 全局store管理
│   ├── utils                  // 全局公用方法
│   └── view                   // 平台组件、全局母版
│        ├── account           // 用户相关信息
│        ├── dashboard         
│        ├── exception         // 错误界面
│        ├── modules           // 配置相关模块内容
│        ├── users             // 注册、登陆
│   ├── App.vue                // 入口页面
│   └── main.js                // 入口 加载组件 初始化等
├── public                     // 第三方等不打包的资源
│   ├── tinymce                // 富文本编辑器
│   ├── index.html             // html模板
│   └── images                 // 静态图片
├── babelrc.config.js          // babelrc 配置文件
├── vue.config.js              // vue 配置文件
├── eslintrc.js                // eslint 配置项
├── .gitignore                 // git 忽略项
└── package.json               // package.json

# 代码提交规范

# Commit message 前缀规范提要

code info
feat:msg 新功能 feature
fix:msg 修复bug
merge:msg merge 信息
docs:msg 文档修改
style:msg 格式，不影响代码运行的变动
refactor: msg 重构即不是新增功能，也不是修改bug的代码变动
test:msg 增加测试
chore:msg 构建过程或辅助工具的变动
rm:msg 删除文件或代码
